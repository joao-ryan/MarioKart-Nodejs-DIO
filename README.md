# 🏎️ Mario Kart Simulator (Node.js)

Simulador de corridas inspirado no universo do Mario Kart, desenvolvido em Node.js e executado diretamente no terminal.  
O projeto foca na construção de uma arquitetura organizada e na simulação de lógica de jogo no backend.

---

## 🚀 Funcionalidades

- 🏁 Sistema de corrida por turnos
- 👤 Múltiplos jogadores
- 🎁 Sistema de itens (boost, banana, etc.)
- ⚔️ Interação entre jogadores
- 📊 Ranking em tempo real
- 🎮 Execução via terminal (CLI)
- 🧩 Arquitetura modular e escalável
- 🔔 Sistema baseado em eventos

---

## 🧠 Tecnologias utilizadas

- Node.js
- JavaScript (ES6+)
- Inquirer (CLI interativo)
- Chalk (estilização no terminal)
